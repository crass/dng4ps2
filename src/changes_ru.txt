Версия 0.2.3beta:
  + Добавлен польский перевод
  + Автодетект типа фотокамеры по данным Exif из Jpeg файла
  + Редактирование профилей фотокамер с возможностью добавления новых профилей
  + Поддержка синонимов названий камер

Версия 0.2.2:
  + Добавлены норвежский, немецкий, испанский, итальянский, китайский и финский переводы
  * Отказался от использования утилиты exiftool
  * Временные файлы preview.tif и preview.jpg больше не создаются, что ускоряет работу программы
  + При AutoISO в dng файл записывается реальное ISO
  + Появилась версия под Linux
  + Добавилась возможность выбирать язык интерфейса
  + Можно конвертировать RAW файлы с JPG расширением
  + Добавлены новые фотокамеры
  - Исправил ошибку, когда если при конвертировании нажималась кнопка "Esc", то окно прогресса закрывалось

Версия 0.2.1:
  * Если есть оригинальный JPEG файл, то он будет использован для превью (для ускорения обработки)
  + Добавился индикатор прошедшего и оставшегося времени преобразования

Версия 0.2.0:
  - Отказался от использования Adobe DNG Convertor
  + Время в метаданных устанавливается корректное даже в том случае, когда метаданные не переносились из JPEG файла
  + Можно добавлять имя фотографа к метаданным
  + Поддержка Canon PS A700

Версия 0.1.1
  + Возможность разбивать DNG файлы на каталоги по датам

Версия 0.1 (Базовая функциональность)
  * Пакетное преобразование в DNG формат с отображением хода преобразования
  * Импорт EXIF снимка из JPEG файла
  * Сжатие DNG файлов без потерь
  * Вставка превью в DNG файл
  * Поддерживаемые фотокамеры: A610, A620, A630, A640, A710 IS, S2 IS, S3 IS
