*** О ПРОГРАММЕ ***

DNG4PS-2 - это программа для пакетного преобразования RAW файлов, сделанных
фотоаппаратами Canon серии PowerShot со вскрытой прошивкой, в формат Digital
Negative (DNG) для последующей обработки в RAW-конвертерах. Название программы
расшифровывается как DNG for PowerShot.


*** КОМПИЛЯЦИЯ ПРОГРАММЫ ***

Для компиляции DNG4PS-2 необходимы следующий софт:

  1. Библиотека wxWidgets (http://www.wxwidgets.org/).

  2. Библиотека boost (http://www.boost.org/)

  3. Среда разработки Code::Blocks (http://www.codeblocks.org/). Скачивать надо
     самый последний nightly-build

  4. Для компиляции под MS Windows необходим MinGW - GCC порт с модулем gettext 
     (http://www.mingw.org/) или Microsoft Visual Studio 2008 (можно использовать 
     бесплатную Express версию - http://www.microsoft.com/express/)

Для компиляции под MS Windows предназначены 
  * проект DNG4PS-2.cbp - для codeblocks,
  * проект DNG4PS-2.vcproj - для Visual Studio

под Linux: 
  * Проект dng4ps2li.cbp - для codeblocks
